# C++ File Encryption Utility

A command-line utility built in C++ for secure file encryption and decryption. This project demonstrates core system programming concepts, file I/O operations, and environment configuration.

## Features
* **File Processing:** Synchronous encryption/decryption of text and data files.
* **Automated Builds:** Configured with a `Makefile` for streamlined compilation and execution.
* **Structured Testing:** Dedicated test directories to validate encryption consistency and accuracy.
* **Environment Management:** Uses `.env` configurations for secure key/parameter handling.

## Tech Stack
* **Language:** C++
* **OS Concepts:** Linux file handling, standard I/O
* **Tools:** Make, Git
